# 🚀 JavaScript `break` Statement Practice

A collection of beginner-friendly JavaScript exercises demonstrating how the `break` statement works in `for` and `while` loops.

---

## 📌 Overview

This repository contains practice programs focused on understanding the `break` statement.

The `break` statement immediately terminates the current loop once a specified condition is met. It is commonly used to stop unnecessary iterations and improve program efficiency.

## 📖 Files Description

### 📄 `break.js`
- Introduction to the `break` statement
- Basic syntax
- Simple examples using loops

### 📄 `break-practice-task-01.js`
- Demonstrate the `break` statement in a `for` loop.

### 📄 `break-practice-task-02.js`
- Stop a `while` loop when the first multiple of **7** is found.

### 📄 `break-practice-task-03.js`
- Search for a target value in a `for` loop and stop once it is found.

### 📄 `break-practice-task-04.js`
- Terminate a `while` loop after encountering the first even number.

### 📄 `break-practice-task-05.js`
- Print the first five odd numbers using a `for` loop with `break`.
