# 🚀 JavaScript `for` Loop Practice

A collection of beginner-friendly JavaScript programs focused on mastering the **`for` loop** through hands-on coding exercises.

---

## 📌 Overview

This repository contains multiple practice problems designed to strengthen the fundamentals of JavaScript `for` loops.

The exercises cover counting, number patterns, mathematical calculations, conditional logic, and basic algorithmic thinking.

# 📖 Practice Tasks

| File | Description |
|------|-------------|
| `for-loop-practice-task-01.js` | Print numbers from **1 to 5** |
| `for-loop-practice-task-02.js` | Print numbers from **1 to 10** |
| `for-loop-practice-task-03.js` | Print numbers from **10 to 1** |
| `for-loop-practice-task-04.js` | Print even numbers from **2 to 20** |
| `for-loop-practice-task-05.js` | Print odd numbers from **1 to 19** |
| `for-loop-practice-task-06.js` | Print numbers from **5 to 15** |
| `for-loop-practice-task-07.js` | Print numbers from **20 down to 10** |
| `for-loop-practice-task-08.js` | Print numbers from **0 to 10** |
| `for-loop-practice-task-09.js` | Print numbers from **50 to 60** |
| `for-loop-practice-task-10.js` | Calculate the sum of numbers from **1 to 100** |
| `for-loop-practice-task-11.js` | Generate the multiplication table of **7** |
| `for-loop-practice-task-12.js` | Calculate the factorial of **5** |
| `for-loop-practice-task-13.js` | Count even and odd numbers from **1 to 100** |
| `for-loop-practice-task-14.js` | Calculate the sum of even numbers from **2 to 50** |
| `for-loop-practice-task-15.js` | Generate the first **100 Fibonacci numbers** |
