# 🚀 JavaScript `while` Loop Practice

A collection of beginner-friendly JavaScript exercises focused on understanding and mastering the **`while` loop** through practical coding challenges.

---

## 📌 Overview

This repository contains multiple JavaScript programs that demonstrate how the `while` loop works in different scenarios.

The exercises cover counting, even & odd numbers, summation, multiplication tables, Fibonacci sequence, and other loop-based problem-solving techniques.
