# 🔄 JavaScript Continue Statement Practice

This repository contains my practice exercises for understanding the **`continue`** statement in JavaScript.

The `continue` statement skips the current iteration of a loop and immediately moves to the next iteration without terminating the loop.

# 📖 Practice Files

## continue.js

**Description**

Notes and basic syntax of the `continue` statement with simple examples.

---

## continue-practice-task-01.js

**Task**

Skip the number **5** while printing numbers from **1 to 10** using the `continue` statement.

---

## continue-practice-task-02.js

**Task**

Print only **odd numbers** using `continue` inside a `for` loop.

---

## continue-practice-task-03.js

**Task**

Skip all numbers that are **multiples of 3** using the `continue` statement.

---

## continue-practice-task-04.js

**Task**

Skip selected numbers based on a condition while iterating through a loop.

---

## continue-practice-task-05.js

**Task**

Print multiples of **5**, but skip all numbers that are also multiples of **10** using `continue`.

---

## continue-practice-task-06.js

**Task**

Find the **first number greater than 50 that is divisible by 7** using the `break` statement.

> This task is included to compare the difference between **continue** and **break**.
