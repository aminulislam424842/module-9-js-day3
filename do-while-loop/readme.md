# 🚀 JavaScript `do...while` Loop Practice

A collection of beginner-friendly JavaScript programs designed to understand and master the **`do...while` loop** through practical exercises.

---

## 📌 Overview

This repository contains hands-on JavaScript practice problems demonstrating how the `do...while` loop executes code **at least once** before checking the condition.

The exercises include counting, even & odd numbers, summation, multiplication tables, factorial calculation, countdowns, multiples, and execution flow.
